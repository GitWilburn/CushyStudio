# VSCode setup

CushyStudio comes with a few vscode extension recommendation (`.vscode/extensions.json`)

Installing them is recommanded for a smooth dev experience

```json
{
    "recommendations": [
        // official vscode support for tailwind
        "bradlc.vscode-tailwindcss",

        // quick related-file swtching
        // configured for tests and other more complex mapping
        "johnathanludwig.fileswitcher",

        // for mdx support
        // mostly usefull if you want to write blog posts or edit documentation
        "unifiedjs.vscode-mdx"
    ]
}
```
